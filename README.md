# 快速开始

pandas的格式如下：

| name  | type  | value | date  |
| ----- | ----- | ----- | ----- |
| 名称1 | 类型1 | 值1   | 日期1 |
| 名称2 | 类型2 | 值2   | 日期2 |

其中“名称”会出现在柱状图的左侧作为Y轴，而“类型-名称”会作为附加信息出现在柱状图上。

类型与柱状图的颜色相关，建议命名为不包含空格或者特殊符号的中英文字符。

值与柱状图的长度绑定，必须是整数或浮点数。

日期建议使用的格式为"YYYY-MM-DD"。
